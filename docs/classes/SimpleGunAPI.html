Class SimpleGunAPI

Simple API wrapper that provides high-level helper methods. For basic operations, use the DataBase instance directly via the database property.

  • arrayToIndexedObject<T extends { id: string | number }>(
        arr: T[],
    ): Record<string, T>

    Convert an array to an indexed object for GunDB storage. GunDB doesn't store arrays natively, so this converts them to objects indexed by ID. Example: [{id: '1', ...}, {id: '2', ...}] => { "1": {...}, "2": {...} }

    Type Parameters

    • T extends { id: string | number }

    Parameters

    Returns Record<string, T>
